# Platform setup

The consumer-facing platform setup for `address_verify` lives in the
package [README](README.md#platform-setup). That section covers:

- iOS `Info.plist` keys (camera, photo library, location-when-in-use).
- iOS minimum deployment target and the `Position.isMocked` caveat.
- Android `AndroidManifest.xml` location permissions.
- Android `minSdkVersion` requirement for ML Kit.
- Geocoding and ML Kit runtime caveats.

This file exists only to redirect contributors who land here from older
docs or commit history. Do not duplicate platform-setup prose here.
